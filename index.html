Core Features

EVERYTHING
YOU NEED

One lightweight extension. Full control over real-time SMS data capture on ivasms.com.

01 / 06
⚡
Zero-Delay Capture

Intercepts Fetch & XHR calls the instant they fire. No polling intervals — data is captured the exact millisecond it arrives from the ivasms.com /test_sms endpoint.

02 / 06
🔴
Start / Stop Recording

Full manual control via the popup UI. Instant kill-switch via Chrome storage listener — no message round-trips required. The content script reacts immediately on state change.

Under the Hood

HOW IT
WORKS

Three parallel capture layers guarantee nothing slips through, from the first network request to the final DOM mutation.

01
Navigate to ivasms.com

Open the portal live page at ivasms.com/portal/live/. The content script activates automatically on page load — no manual injection needed.

02
Click Start Recording

Click the extension icon and hit "Start Recording". An instant storage signal fires — the content script starts all three capture layers simultaneously with no round-trip delay.

03
Data Captured Instantly

The Fetch interceptor hooks the /portal/live/test_sms endpoint. XHR monitoring catches fallback requests. MutationObserver watches for new DOM rows — all fire at the exact moment data arrives.

04
Search, Filter & Export

Use the live search to filter by phone number, sender, or message content — with optional regex. When you're done, export everything to JSON with one click.

Installation

      
UP IN
MINUTES

      
No Chrome Web Store listing required. Load the extension directly in Developer Mode — takes under two minutes.

    


    


      

        
1

        
📦

        
Download ZIP

        
Click the download button to get the extension package. Extract the ZIP to any folder on your computer.

      


      

        
2

        
🧩

        
Open Extensions

        
In Chrome, navigate to chrome://extensions/ and toggle on Developer mode in the top-right corner.

      


      

        
3

        
📂

        
Load Unpacked

        
Click "Load unpacked" and select the extracted folder. The Live Range Scraper icon will appear in your Chrome toolbar immediately.

Visit ivasms.com/portal/live/, click the extension icon, hit Start Recording and watch data stream in live.
